# TRETRA. — Personal Portfolio

> I Build At The Edge.

Where AI, quantitative trading, and education collide.

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- Three.js (3D hero scene)
- Vite (dev server)
- Bilingual support (EN / 中文)

## Features

- Three.js icosahedron with particles and mouse-reactive rotation
- 3D card tilt effect on project cards
- Custom cursor with mix-blend-mode difference
- Scroll-triggered reveal animations
- Parallax hero typography
- Responsive design with mobile menu

## Development

```bash
npm install
npx vite
```
